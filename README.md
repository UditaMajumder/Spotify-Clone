# 🎵 Spotify Clone

A simple Spotify Web UI Clone built using HTML and CSS.

## 🚀 Live Demo
🔗 https://uditamajumder.github.io/Spotify-Clone/

## 💻 GitHub Repository
🔗 https://github.com/UditaMajumder/Spotify-Clone

## ✨ Features
- Sidebar navigation
- Music player UI
- Album cards layout
- Volume control slider
- Clean and modern design

## 🛠️ Technologies Used
- HTML
- CSS (Flexbox, styling)

## 📚 What I Learned
- How to structure a webpage using HTML
- Using Flexbox for layout
- Styling components with CSS
- Building a real-world UI design
